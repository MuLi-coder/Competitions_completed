# 可参加竞赛汇总
## 一、 全国大学生数学建模竞赛（CUMCM/国赛）
三人小队，完成一道题的建模计算出论文的完整过程
### 时间
- 大一下 参加体验，时间等待通知
- 大二下

**九月正式竞赛**
### 技能
- 建模手：根据往年题库中的题目，阅读往年优秀论文，掌握常见数学模型及分析方法

- 编程手：和建模手同步学习并理解，建立模型对应的代码库以便调整，MATLAB/Pyhton均可，需具备代码绘图能力

- 写作手：和前面两人同步学习阅读论文，学习框架及表述，积累常见句式，学习LaTeX，做到可以差不多的美观排版

> **最终建议：三者同步学习，三个方向的能力都掌握一点，便于共同发展，学习新技能**

## 二、 中国大学生程序设计竞赛（CCPC）
三人小队，三人一台电脑限时完成10-13道算法题，竞速赛
### 时间
大二上8-9月**线上竞赛**

### 技能
- 对于算法和编程实现均熟悉  
- 首选语言C++

### 知识

|知识领域|具体内容|
|------|-------|
|基础算法|枚举、模拟、贪心、二分、排序、搜索|
|数据结构|栈/队列/堆、并查集、线段树、树状数组、平衡树|
|图论|最短路、最小生成树、拓扑排序、强连通分量、网络流|
|动态规划|背包问题、区间DP、树形DP、状压DP|
|数学|数论（素数、GCD、同余）、组合数学、博弈论、计算几何|
|字符串|KMP、AC自动机、后缀数组、Manacher|

评价：如果三个人觉得LeetCode中等难度题都已经做得很溜了，那才刚摸到CCPC的门槛。

> **最终建议：三人均努力学习算法及其实现，作为数学竞赛的拓展，为中国版的ICPC**

## 三、 中国大学生计算机设计大赛(4C大赛)
2-5人团队，成果：项目作品+演示视频+答辩PPT
### 时间
- 大二寒假期间直接带着完整作品进行校园报名  

**三月份校赛**
### 技能
- 开发一个完整的项目作品（软件/app/游戏等）

> **最终建议：可以在大一下到大二上寒假的期间慢慢完成一个Web开发的项目，期间慢慢在学习其他竞赛内容的过程中会有同步的学习内容**

## 四、 ACM国际大学生程序设计竞赛（ICPC）
三人小队，共用一台电脑，五小时完成10-13道算法题
### 时间
- 大二下 3月校内选3支队伍
### 技能
- 全英文题目
- 首选C++

> **最终建议：是CCPC的国际版，算法和编程实现能力要求高，由于是国际赛事，数学英语的能力要过关**

## RoboMaster 战队内部详细分工

| 组别 | 核心任务 | 所需技能 | 专业背景建议 |
| :--- | :--- | :--- | :--- |
| **机械组** | 机器人结构设计、强度校核、加工装配、维护修理 | 机械设计（SolidWorks）、材料选型、仿真分析 | 机械/车辆/材料 |
| **智能组** | **视觉**：自动瞄准（C++/Python，基于Linux图像处理、神经网络）<br>**导航**：哨兵机器人自主路径规划 | 计算机视觉、深度学习、ROS、C++/Python | 计算机/自动化/软工 |
| **电控组** | **硬件**：电路设计、PCB绘制（立创EDA）、供电系统<br>**软件**：STM32嵌入式开发（HAL/RTOS）、PID控制算法、通信协议（CAN/UART） | 嵌入式、电路设计、自动控制 | 电子/电气/自动化 |
| **运营组** | 财务管理、宣传推送、周边设计、后勤保障 | 文案、设计、组织协调 | 不限（经管/设计/传媒） |

> **最终建议：只能进入社团参与，今年的报名已经截止，此外，社团位于仙林，对苏州的学生十分不利，另外鉴于南京大学的战队刚刚组建，其实只能参与RMUL入门级比赛，并非真正意义上的RoboMaster比赛**

## 六、 中国国际大学生创新大赛（互联网+大赛）
开公司创业的前一步
### 时间
- 大一下 了解
- 大二下 寒假可携带完成的项目报名参与
### 参赛组别
本科生创意组：有创意、有原型或服务模式，但尚未注册公司
### 核心材料 
| 材料 | 要求 | 说明 |
| :--- | :--- | :--- |
| **商业计划书** | 不超过20MB | **核心材料！** 包括：产品/服务介绍、市场分析、商业模式、营销策略、财务分析、风险控制、团队介绍等 |
| **项目展示PPT** | 不超过20MB | **路演PPT**，要美观、逻辑清晰、数据可视化 |
| **项目展示视频** | 1-3分钟（部分学校要求） | 展示产品/技术/应用场景 |
| **佐证材料** | 附件形式 | 专利、软著、论文、检测报告、获奖证书、营业执照等 |
| **参赛报名表** | 按学校要求填写 | 校内报名用 |

**特别注意**：商业计划书和PPT必须严格按照 **评审规则** 的得分点来制作。评审维度包括：创新性、商业价值、团队实力、社会效益等。

> **最终建议：涉及创业的相关内容，需要有企业经营者或者商科同学参与**

# 最终阐述
### 目前看来，有可能真正准备并且实现的竞赛有如下几个：
- ***全国大学生数学建模竞赛***（最可能）
- CCPC（难度较大）
- 4C大赛（需要提前准备项目）
- 互联网+（商业产品的项目）  

不建议的：
- ICPC：国际版的CCPC,层层选拔，难度更上一层
- RoboMaster：地域问题不利于参加，战队刚起步参加的入门赛含金量不够高，但主要是前者
