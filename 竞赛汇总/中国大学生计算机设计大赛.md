# 中国大学生计算机设计大赛

A类竞赛，保研加分，进入国赛保研有强竞争力

## 基本信息

- **官网链接**：http://jsjds.blcu.edu.cn/wjss.htm
- **举办方**：中国高等教育协会等
- **性质/含金量**：A类竞赛，保研加分，进入国赛保研有强竞争力

## 竞赛流程

**备赛/组队（2~9人）（1~3月）** → **校赛/省赛（3~5月）** → **5月30日国赛上报截止** → **国赛决赛（7~8月）** → **结果发布（9~10月）**

## 竞赛内容

这个竞赛涉及到的范围非常广，总共有十一类：

- **1~5类**（不限主题）：
  - 软件应用与开发
  - 微课与AI辅助教学
  - 物联网应用
  - 大数据应用
  - 人工智能应用

- **6~11类**（强调AI与艺术等其他专业的跨专业结合，根据比赛主题完成相关大类开发）：
  - 如2026年主题是**中国古代建筑成就**，其中有一个大类是**AI+信息可视化设计**，就要在这个大类下完成一个作品

## 必备技能

由于这项比赛涉及到的范围非常广，允许的组队人数非常多，所以灵活度很高：

- 可以自己参加，作为专业知识的巩固，学习一些新的技能去完成一些小的项目开发（1~5大类）
- 也可以和别的专业同学组队去完成一些跨学科项目

## 专业课相关度

与软件工程、人工智能、大数据等专业课高度相关。
