# 中国大学生程序设计竞赛（CCPC）

国内顶级程序设计竞赛，CCPC是国内**含金量最高的算法竞赛之一**

## 赛制

CCPC采用**多站选拔+全国总决赛**的赛制：

- **网络选拔赛**：所有报名队伍参加线上预选，筛选出进入各分站赛的队伍
- **分站赛（区域赛）**：在全国多个城市举办，各站独立评奖
- **全国总决赛**：各分站赛优秀队伍晋级全国总决赛

## 时间

- **网络选拔赛**：**8月-9月**（线上）
- **分站赛（区域赛）**：**9月-11月**（线下，多个城市巡回）
- **全国总决赛**：**11月-12月**（线下）

每年具体时间以CCPC官网发布的赛历为准。

## 形式

- 以**团队赛**形式进行，每支团队由**3名学生**组成
- 比赛时长：**5小时**
- 题目数量：**10-13题**（英文题面）
- 编程语言：**C/C++、Java、Python**（主流OJ均支持）
- 评判方式：实时评测，每题通过即时获得彩色气球（ACM赛制）

### 竞赛规则
- **ACM赛制**：每道题通过即获得该题分数，未通过不扣分；通过题数多者排名靠前，同题数时罚时少者胜
- **封榜机制**：比赛最后1小时榜单冻结，不再显示实时排名
- **允许携带**：纸质参考资料（书籍、打印资料），**禁止携带电子设备**

## 奖项设置

| 奖项 | 说明 |
|------|------|
| **分站赛金奖** | 每站前10%左右（视参赛队伍数浮动） |
| **分站赛银奖** | 每站前20%-30%左右 |
| **分站赛铜奖** | 每站前40%-50%左右 |
| **全国总决赛金奖** | 总决赛队伍中的最高荣誉 |



## 提交材料

- **线上评测**：在比赛系统中提交源代码，实时获得评测结果
- **无需额外提交**：论文、报告、文档（纯算法竞赛，只看解题能力）

## 知识基础

- **基础算法**：枚举、模拟、贪心、二分、搜索（DFS/BFS）
- **数据结构**：栈、队列、链表、树、并查集、堆、线段树、树状数组
- **图论**：最短路、最小生成树、拓扑排序、强连通分量、网络流
- **动态规划**：背包问题、区间DP、树形DP、状压DP
- **数学**：数论（素数、GCD、同余）、组合数学、计算几何、概率论
- **字符串**：KMP、AC自动机、后缀数组、哈希
- **进阶**：莫队算法、CDQ分治、FFT/NTT、平衡树等
